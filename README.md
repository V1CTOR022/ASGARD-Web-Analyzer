# ASGARD Web Analyzer 🌐

O **ASGARD Web Analyzer** é uma ferramenta simples de terminal feita em Python para analisar informações básicas de sites através de uma URL.

O projeto foi criado com foco em aprendizado de requisições HTTP, análise de HTML e organização de código em Python.

---

## 🚀 Funcionalidades

- 🔗 Acessa qualquer URL informada
- ⏱️ Mede tempo de resposta do site
- 🌐 Identifica IP do domínio
- 🔒 Verifica se o site usa HTTPS
- 🧾 Identifica servidor (quando disponível)
- 📊 Conta links e imagens da página
- 🏷️ Extrai o título do site
- ⚠️ Detecta possíveis páginas de login
- ⚠️ Detecta presença de campos de senha
- 🎨 Interface colorida no terminal

---

## 🛠️ Tecnologias utilizadas

- Python 3
- urllib
- socket
- re (expressões regulares)
- colorama
- pyfiglet
- time
